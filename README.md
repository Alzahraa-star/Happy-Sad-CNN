## CNN Emotion Classification Project
This project uses a Convolutional Neural Network (CNN) to classify facial emotions into Happy and Sad categories using the FER2013 dataset.
## Dataset:
FER2013 facial expression dataset from Kaggle. Images were resized to 48x48 and split into training and testing sets.
## Tools:
•	Python

•	TensorFlow

•	Keras

•	Google Colab
## Model:
A CNN model was built using convolution, pooling, dense, and dropout layers.
## Results:
•	Training Accuracy: 94%

•	Validation Accuracy: 84%
